# 项目名称：GGWP游戏介绍网站
## 作者介绍
* 小组名称：快乐肥宅/GGWP  
* 开发时间：2018.7.14-2018.7.15
## 项目介绍
* 项目内容：包含主页，用户登录界面，用户注册界面，游戏详情界面  
* 项目功能：实现用户登录与注册，游戏详情的查看，游戏的评论、评分功能
## 开发软件
* NetBeans、MySQL（数据库连接）、GitHub（小组合作)  
## 使用方法
* 使用软件：NetBeans，mysql  
* 运行服务器：Apache Tomcat
